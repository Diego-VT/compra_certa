# UI/UX Guidelines - CompraCerta

## Identidade Visual

O CompraCerta deve transmitir organizacao, clareza e confianca. A interface deve ser simples, direta e adequada a uso recorrente.

## Cores

- Usar Material Design 3 como base.
- Manter suporte a tema claro e escuro.
- Preservar contraste adequado.
- Evitar paletas pesadas ou com baixa legibilidade.

## Tipografia

- Usar tipografia padrao do Material Design.
- Priorizar legibilidade.
- Evitar textos longos em botoes e componentes compactos.

## Espacamentos

- Usar espacamentos consistentes.
- Separar grupos de informacao por hierarquia visual.
- Evitar telas densas demais em dispositivos pequenos.

## Componentes

- Formularios devem indicar campos obrigatorios.
- Listagens devem permitir busca e filtros quando houver volume.
- Acoes destrutivas devem exigir cuidado.
- Produtos devem usar status claro para ativo/inativo e, futuramente, estoque.

## Icones

- Usar icones consistentes com Material Design.
- Icones devem reforcar a acao, nao substituir clareza quando houver duvida.

## Acessibilidade

- Garantir contraste.
- Evitar informacao apenas por cor.
- Usar textos claros.
- Preservar tamanhos tocaveis adequados.

## Responsividade

- Prioridade inicial: smartphones Android.
- Layouts devem se adaptar a telas pequenas.
- Evitar widgets gigantes e rolagens confusas.

## Boas Praticas de Usabilidade

- Menos passos para tarefas frequentes.
- Busca sempre que a lista puder crescer.
- Feedback visual para salvar, erro e vazio.
- Estados vazios devem orientar o usuario.
- Fluxos offline nao devem depender de internet.
